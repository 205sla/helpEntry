복잡한 질문은 밴드에 올려 주시면 자세한 답변이 편합니다. 댓글이 아닌 ‘게시글’로 질문을 작성해 주세요.

바로 가기
안내: 밴드에서는 댓글이 아닌 게시글로 질문을 올려 주세요. 캡처 이미지는 텍스트가 선명한 화면 캡처로 첨부하면 검토가 훨씬 빨라집니다.

질문 전 꼭 읽어주세요

  1. 코드를 함께 올려 주세요.
    캡처한 이미지나 엔트리 작품 링크가 좋아요. 사진으로 찍은 화면보다 화면 캡처가 더 또렷해요.
  2. 무엇을 만들고 싶었는지 적어 주세요.
    기대한 동작을 한 줄로 써 보아요. 참고한 영상/글이 있다면 링크도 함께요.
  3. 이미 해 본 방법을 알려 주세요.
    시도한 것과 결과를 적어 주면, 같은 방법을 다시 안내하지 않게 도와줘요.
  4. 오류가 언제, 어디서 생겼는지 알려 주세요.
    어떤 화면/오브젝트에서 문제가 생겼는지 쓰고, 스크린샷(또는 짧은 영상)을 올려 주세요.
  5. 해결되면 한마디만!
    먼저 해결됐다면 알려 주세요. 어떻게 해결했는지 남기면 다음 친구들에게 큰 도움이 돼요.

카톡이 좋을까요, 밴드가 좋을까요?

작은 질문은 카톡, 자세한 문제/버그는 밴드 글을 추천해요.

코드 캡처, 이렇게 해요

Windows

  • Win + Shift + S 를 누른 뒤, 캡처하고 싶은 영역만 드래그해요. 클립보드에 저장되니 그림판/포토 앱에서 붙여넣기 후 PNG로 저장하면 끝!

macOS

  • ⌘ + Shift + 4 를 누르고, 원하는 영역만 드래그해요. 자동으로 바탕화면에 저장돼요. 필요하면 파일 이름만 바꿔서 올리면 돼요.
