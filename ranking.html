랭킹 만드는 법

게임 점수를 모아 순위표를 보여주는 방법을 쉬운 말로 정리했어요. 아래 순서대로 천천히 따라와요.

요약: 이름/점수 저장 → 점수 기준 정렬 → 상위 N명 보여주기

영상으로 보기

유튜브로 보기: https://youtu.be/ksnSnkSKIY4

준비물

  • 변수: 이름, 점수
  • 리스트: 학생이름, 학생점수
  • 화면에 보여줄 순위표 영역

이 페이지에서 배우는 것

  • 점수와 이름을 리스트에 넣는 법
  • 점수 기준으로 정렬하는 법
  • 상위 N명을 화면에 출력하는 법

단계별로 따라하기

  1. 점수 저장
    게임이 끝났을 때 이름과 점수를 각각 리스트에 넣어요. (예: 학생이름에 이름 추가하기, 학생점수에 점수 추가하기)
  2. 정렬
    점수가 큰 순서대로 정렬해요. 동점이면 이름으로 한 번 더 정렬하면 깔끔해요.
  3. 상위 N명
    5명 또는 10명까지만 화면에 보여줘요. 너무 많으면 글자가 작아져요.
  4. 화면에 출력
    1등부터 차례대로 "이름 - 점수" 모양으로 적어줘요. 줄바꿈을 써서 보기 좋게 만드세요.

FAQ

동점이 많으면 순위가 이상해 보여요.

점수로 먼저 정렬한 뒤, 이름으로 한 번 더 정렬해요. 또는 저장한 시간을 써서 더 먼저 저장한 친구를 위로 올릴 수도 있어요.

리스트가 너무 길어요.

상위 5명만 보여주고, 나머지는 “더보기” 버튼으로 넘겨요. 화면이 좁은 모바일에서 특히 좋아요.

점수가 0점일 때도 보여줄까요?

원하면 0점은 빼고 보여줘도 돼요. 규칙을 미리 정하고 모두에게 똑같이 적용해요.
